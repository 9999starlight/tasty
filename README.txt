Full stack (MEVN) app
Tools/libraries: 
Client - Vue.js, Vuex, Vue router; SCSS - responsive layout, flexbox & grid. vuex-persistedstate, vue-axios, @fortawesome
Server - Node/Express.js, Mongoose, JWT auth. Middlewares: @hapi/joi validator, Multer. Database: Mongodb Atlas. Image upload: Cloudinary
